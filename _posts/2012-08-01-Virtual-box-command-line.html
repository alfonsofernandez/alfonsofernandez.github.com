--- layout: post title: "Virtualbox command line virtual machine boot" published: true meta: _edit_last: "1" tags: - GNU/Linux type: post status: publish categories: - gnu-linux - mac - virtualbox --- Some usefull virtualbox commands To get a list of all your virtual machine with name and id 
VBoxManage list vms
You'll get something like
"UbuntuServer" {5a5f1895-b73b-4763-8cd2-54642a9661d1}
"DevEnv_V2.2" {17bb6c14-74eb-4ca2-97c8-b68f1d4b2d05}
Then you can boot a specific virtual machine with the following command including the id of the vm. 
	VBoxManage startvm 5a5f1895-b73b-4763-8cd2-54642a9661d1 --type headless


	VBoxManage controlvm 5a5f1895-b73b-4763-8cd2-54642a9661d6 poweroff
